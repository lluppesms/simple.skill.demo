# Hello World
Use the `hello-world` skill to get a friendly greeting whenever the user types the phrase "hello world". 
Run the script to get more info about the system and the user, then respond with a cheerful "Hello there, <$User>!" message in ASCII art.

Run the PowerShell script located at `.github/skills/hello-world/scripts/Get-CurrentUser.ps1` to retrieve system and user information. Use the output to enrich your greeting.
